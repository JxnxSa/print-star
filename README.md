## วิธีการ Run  
**run บน Terminal** :  
 1. run คำสั่ง ```git clone https://github.com/JxnxSa/print-star.git```
 2. run คำสั่ง ```npm install```
 3. run คำสั่ง ```npm run dev```
    
**run บน web** : https://print-star.netlify.app
